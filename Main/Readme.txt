	□動作環境
	　Microsoft Windows XP SP2以上
	　Microsoft .NET Framework 2.0以上
	
	もしくは
	
	　Microsoft Windows Vista SP1以上

	これ以外のOSや，64bit版のWindows OSでの動作保証はしません．

	□開発環境
	　Microsoft Windows XP Vista Home Premium Service Pack 1 32bit版
	　Microsoft Visual Basice 2008 + Microsoft .NETFramework 2.0

	□対応セーブデータ
	■sav ファイル
	VisualBoyAdvance のバッテリーファイル
	もしくはブートケーブルなどで読み出されたセーブデータ。

	■xps ファイル
	GBA用 ProActionReplayで読み出されたセーブデータ。
	動作保証外です。動かなかったとしても対応する気はありません。

	■sps ファイル
	VisualBoyAdvance のGameSharkSnapShotファイルもしくは
	海外版GBA用PARで読み出されたセーブデータ。

	□使い方
	気合で何とかしましょう。

	□注意とか
	■フォルダの編集機能
	持っていないフォルダに対しては使わないことを推奨
	■フォルダの数
	増やさない方が無難
	■フォルダのところの補正ボタン
	フォルダ内のチップ数よりリュックの中のチップ数が少ない場合は
	リュックの中のチップ数を増やす。これをやらないと不正なチップエラーがｗ

	□著作権
	■このツールの配布について
	アーカイブ内を改変していなければ再配布を許可します。
	アーカイブ内を改変（ファイルの追加、削除）した形での再配布は禁止します。
	ただし、ソフトウェアを配布することで何らかの対価を得るような方法で
	配布する場合はこの限りではありません。配布に使ったメディアの代金などは可
